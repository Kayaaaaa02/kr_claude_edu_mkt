# 주간 매출 대시보드 설계안

## 목적
매주 월요일, 채널·카테고리 성과를 한눈에 점검한다.
- 금주(2026-05-11~17) vs 전년 동주(2025-05-12~18) **YoY 비교**
- 어느 채널/카테고리가 잘 팔렸는지, TOP5 제품은 무엇인지 한눈에 본다.

## 데이터
- 파일: `MLB_Sample_data.xlsx`
- 시트: 개요 / 채널별매출비교 / 카테고리별매출비교 / 상위카테고리_TOP5제품 / 인사이트
- 주요 컬럼: `금주_판매액`, `전년_판매액`, `판매액_신장율(%)` — 판매수량도 동일 구조

## 화면 스케치

```
┌──────────────────────────────────────────────────────────────┐
│  주간 매출 대시보드                       2026-05-11 ~ 05-17 │
├──────────────────────────────────────────────────────────────┤
│  [KPI] 총매출 금주 | 전년 동주 | 신장율                      │
├───────────────────────────┬──────────────────────────────────┤
│  채널별 판매액 & 신장율   │  카테고리별 판매액 & 신장율      │
│  (가로 막대)              │  (가로 막대)                     │
├───────────────────────────┴──────────────────────────────────┤
│  상위 카테고리 TOP5 제품 테이블                              │
├──────────────────────────────────────────────────────────────┤
│  인사이트 요약 카드                                          │
└──────────────────────────────────────────────────────────────┘
```

- **상단**: 총매출 KPI 카드 (금주 / 전년 / 신장율)
- **중단**: 채널별·카테고리별 가로 막대 (판매액 + 신장율)
- **하단**: 상위 카테고리 TOP5 제품 테이블 + 인사이트 요약 카드
